  
# ⚡ Electric Dipole Field Simulation

A Python-based simulation of electric field lines and equipotential contours generated by a pair of point charges (electric dipole).

---

## 🧠 Overview

This project visualizes the electric field and potential distribution of an electric dipole system using numerical computation.

It demonstrates how fundamental electrostatics principles—such as Coulomb’s law and superposition—lead to structured field patterns and symmetry.

---

## 🔬 Physics Concepts

- Coulomb’s Law  
- Electric Field (vector field)  
- Electric Potential (scalar field)  
- Superposition Principle  
- Electric Dipole Symmetry  

---

## 📊 Features

- Visualizes electric field lines using stream plots  
- Displays equipotential contours (including zero-potential symmetry line)  
- Supports multiple charge configurations  
- Combines vector field and scalar field visualization in a single plot  

---

## 🧮 Methodology

The simulation is based on standard electrostatic equations:

- Electric field:

  E = q * r / |r|³

- Electric potential:

  V = q / |r|

The total field is computed by summing contributions from all charges (superposition).
